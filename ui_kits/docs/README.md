# Docs UI Kit — Bridge Burners

Methodology handbook, taxonomy reference, contributor docs.

## Components

- `DocsShell.jsx` — three-pane: sidebar (sections) + article + on-page TOC
- `DocsSidebar.jsx` — nested nav with versioning + search
- `Article.jsx` — heading, lede, prose with code samples, callouts, footnotes
- `Callout.jsx` — bordered slabs for warning/note/sourced
