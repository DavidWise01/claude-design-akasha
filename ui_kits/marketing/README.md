# Marketing UI Kit — Bridge Burners

Public-facing site: identity, manifesto, reports index, contact.

## Components

- `Header.jsx` — sticky top bar with logo lockup + nav + CTA
- `Hero.jsx` — massive Archivo headline, ember underline, ochre dossier badge
- `ManifestoBand.jsx` — full-bleed ink band with §-numbered tenets
- `ReportCard.jsx` — bone card with stamp shadow, eyebrow + headline + meta
- `ReportsGrid.jsx` — featured + grid of 6 reports
- `StatBlock.jsx` — numeric facts (47/50, 12 weeks, etc.)
- `Newsletter.jsx` — single-field mono input with ember CTA
- `Footer.jsx` — dark moss footer with sitemap and colophon

## Screens (in index.html)

A single scrolling marketing home: Header → Hero → StatBlock → ManifestoBand → ReportsGrid → Newsletter → Footer.
